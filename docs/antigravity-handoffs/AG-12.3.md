## 미실행 브라우저 테스트 및 한계점
- **localStorage 한계점**: 
  - 본 기능(템플릿, 자동완성, 임시저장)은 서버 DB가 아닌 현재 접속 중인 사용자의 브라우저 로컬 저장소에만 저장됩니다. 따라서 브라우저를 변경하거나, 시크릿 모드를 닫거나, 캐시를 지우면 데이터가 소실됩니다. 
  - 최근 검색어는 100건까지만 누적 저장되며 넘어가면 오래된 내역부터 버려집니다. 
- **보안**: 
  - 민감한 개인정보나 서버 전체의 기록은 일절 수집되지 않고 사용자가 텍스트 폼에 직접 치고 저장 버튼을 눌러 확정한 "품명"만 로컬에 남아 제안됩니다.
- **테스트 미실행**:
  - `window.prompt`, `window.confirm`, 자동완성 UI(`<datalist>`) 동작 등, 브라우저가 직접 그려주는 상호작용 관련 테스트는 Vitest 단위 테스트로는 한계가 있어 수동으로 확인해야 하며 Playwright 등 E2E 테스트는 별도 환경 구성이 필요하므로 미실행되었습니다.

## 미확인 업무 규칙
- 브라우저 간 설정 동기화 기능(클라우드 템플릿)의 필요성.
- 공용 PC 사용자의 경우 `localStorage`를 통한 이전 사용 내역 유출에 대한 우려 및 로그아웃 시 클리어 정책 필요 여부.
